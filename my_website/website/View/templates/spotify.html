My Spotify Overlay Project

This project is an overlay for spotify, allowing the user to do other tasks without having to alt tab to adjust spotify playback. I built it using PyQt5, a graphic user interface toolkit, and pyCurl, a python interface allowing one to make http requests. PyQt5 handles all of the graphical sides to the application such as the placement of the buttons, while pyCurl handles all of the API calls. This project can be found on my github. Github Repository
