My Spotify Overlay Project

This project is an overlay for spotify, allowing the user to do other tasks without having to alt tab to adjust spotify playback. This app can control spotify on all your devices! I built it using PyQt5, a graphic user interface toolkit, and pyCurl, a python interface allowing one to make http requests. This project can be found on my github. Github Repository
